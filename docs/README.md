## CLI

- **REPL Mode**: An empty arguments or flags will put you into REPL mode.

- **Run Script File**: You may use `run` command to provide a script file.

  ```bash
    interpreter run -i my-script.iad
    interpreter run -i my-script.iad -o out
  ```
